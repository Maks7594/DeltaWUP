# DeltaWUP

DeltaWUP is a port of DELTARUNE to Wii U, powered by LOVE Potion.

## Controls

D-Pad, Left Analog - move
A - interact, confirm selections, progress through textboxes
B - cancel, run, skip typing in textboxes
X - menu
Start (+) - quit
